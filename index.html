AutoClicker

Short Description: Windows Forms application that lets users visually build automation scripts using Click, Type and Wait commands and run them; scripts are persisted as JSON files under Desktop/JSONs.

Key Features:

  • Visual script builder: add Click (screen coordinates), Type (text or modifier keys), and Wait (seconds) steps with a simple UI and real-time cursor coordinate capture.
  • Script management & execution: save/load/delete scripts as JSON, dynamically list available scripts, run scripts asynchronously with optional looping and cancellation (End key), and simulate mouse/keyboard via P/Invoke and SendKeys.

Technologies Used: C# (WinForms on .NET 10), System.Text.Json for serialization, P/Invoke to user32.dll (GetCursorPos, mouse_event), SendKeys, and file I/O for JSON persistence.

Outcome: Users can create, persist, share, and run repeatable click/type/wait sequences to automate repetitive desktop tasks; scripts are human-readable and portable JSON files.

ImprovementApp

Short Description: Lightweight Blazor Interactive Server application to collect, review, and persist continuous improvement suggestions via a simple web form and list view.

Key Features:

  • Submit suggestions with a typed model using `EditForm` (default date set to today).
  • View, annotate, mark winners, and save suggestions; persisted to JSON through the injected `WriteToJSON` service.

Technologies Used: .NET 8, C# 12, Blazor Interactive Server Components, Razor Components, IJSRuntime (JS interop), Bootstrap, JSON-based persistence.

Outcome: A simple internal tool for capturing and managing improvement ideas with minimal setup and file-based persistence for easy inspection and export.

NonMetal Helper

Short Description: A Windows Forms utility that streamlines pricing lookup, quote note generation, and quick access to commonly used files and websites for non-metal product lines.

Key Features:

  • Parses FSG part numbers, populates a pricing Excel matrix, reads calculated price tiers, and copies formatted price/quote blocks to the clipboard for fast quoting.
  • Creates configurable quick-access buttons from a JSON config and builds customizable quote notes; includes robust Excel interop cleanup (PID tracking + COM release) to avoid lingering processes.

Technologies Used: C# (.NET 8), Windows Forms, Microsoft.Office.Interop.Excel, System.Text.Json for configuration, Windows clipboard & process APIs, P/Invoke for PID retrieval.

Outcome: Faster, repeatable quoting and pricing workflows with reduced manual Excel errors and reliable Excel process cleanup; consistent customer quotes and easy extensibility via JSON-driven buttons.
